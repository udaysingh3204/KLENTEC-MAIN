## 🌐 **DEPLOY TO PRODUCTION (2 MINUTES)**

### **Recommended: Use Netlify**

1. **Create account**: https://netlify.com (free)
2. **Connect GitHub**: Authorize `udaysingh3204/KLENTEC-MAIN`
3. **Add Environment Variables**:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   - VITE_EMAILJS_PUBLIC_KEY
   - VITE_EMAILJS_SERVICE_HELLO
   - VITE_EMAILJS_SERVICE_ADMIN
   - VITE_EMAILJS_TEMPLATE_INQUIRY
   - VITE_EMAILJS_TEMPLATE_ADMIN
4. **Deploy!** Click deploy button
5. **Custom Domain**: Point www.klentec.com to Netlify DNS

**That's literally all you need to do.**

→ **Full instructions**: Read `DEPLOYMENT_GUIDE.md`
